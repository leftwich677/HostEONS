## 中文环境配置核心方法

### 方法一：通过扩展安装语言包
1. 使用快捷键 **Ctrl+Shift+X** 唤醒扩展管理面板
2. 搜索关键词："Chinese Language Pack" 或 "中文语言包"
3. 安装官方认证的扩展程序（建议选择星标最高的模块）
4. 重启编辑器完成语言切换

### 方法二：配置文件直接修改
1. 通过 **Ctrl+Shift+P** 调出命令面板
2. 输入："Configure Language" 启动语言设置窗口
3. 在语言参数框输入：
   json
   {
     "locale": "zh-CN"
   }
   
4. 保存修改后重启 IDE
